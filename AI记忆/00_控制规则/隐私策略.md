# 隐私策略

## 禁止长期保存

不要把以下内容写入长期记忆：

- 密码、token、密钥、证书、私钥。
- 账号凭证、身份信息、支付信息。
- 原始日记全文。
- 未脱敏聊天记录。
- 公司或客户敏感材料。
- 无法确认授权的第三方内容。

## 禁止默认读取

路径或文件名包含以下关键词时，不要默认读取、索引、摘要或候选化：

```text
password
credentials
secrets
token
keychain
private
账号密码
```

## 可保存的替代形式

可以保存脱敏后的稳定规则：

- “用户偏好简洁中文回复”
- “某项目指标分母必须先确认”
- “某类错误需要先做口径审计”

不要保存原始敏感证据。
