# 公开发布安全清单

把记忆库模板上传到公开 GitHub 前，至少检查以下项目：

- 不包含真实用户画像、个人偏好、公司项目记忆、业务数据或复盘结论。
- 不包含本机真实绝对路径，除非是文档示例且已做脱敏。
- 不包含 API key、token、密码、私钥、证书、账号凭证。
- 不包含原始日记、聊天记录、导出数据、截图、PDF 或附件。
- 不包含客户、同事、公司内部系统、数据库表的敏感解释。
- `.gitignore` 已忽略来源归档、日常记录、附件和候选区。
- 发布前运行全文搜索：`password`、`credentials`、`secrets`、`token`、`账号密码`、真实用户名、真实项目名。

建议：公开仓库只放模板和规则；真实记忆库放私有仓库、本地磁盘或同步盘。
