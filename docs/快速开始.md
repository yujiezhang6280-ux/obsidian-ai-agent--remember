# 快速开始

## 1. 复制模板

把本仓库复制到 Obsidian vault 里，例如：

```text
<你的 Obsidian vault 路径>/Agent记忆库
```

## 2. 替换路径占位符

全局搜索：

```text
<你的 Obsidian vault 路径>
```

替换成你的本机 Obsidian vault 路径。

## 3. 配置 Agent 入口

不同 Agent 的启动文件不同。最稳的做法是让每个 Agent 的系统入口指向：

```text
<你的 Obsidian vault 路径>/Agent记忆库/说明.md
<你的 Obsidian vault 路径>/Agent记忆库/AGENTS.md
<你的 Obsidian vault 路径>/Agent记忆库/CLAUDE.md
```

如果某个 Agent 有自己的全局配置文件，把对应入口内容复制进去，并保留 Obsidian 记忆库路径。

## 4. 创建新项目记忆

复制：

```text
AI记忆/30_项目记忆/_模板/项目记忆模板/
```

到：

```text
AI记忆/30_项目记忆/进行中/<项目名>/
```

然后先填写：

- `_项目记忆/00-关键上下文.md`
- `_项目记忆/99-索引.md`

## 5. 写入原则

- 不确定是否长期有用时，先写候选。
- 项目局部事实写项目记忆。
- 跨项目稳定规则再提升到核心记忆。
- 来源、日记、附件默认不提升。
